# 探测器几何尺寸说明

基于 `DetectorConstruction.cc` 文件的几何参数总结

## 1. 探测器阵列参数

### 像素数量
- **像素总数**：128个（`DetPixNum = 128`）
- **像素间距**：0.8 mm（`GapPeriod`）
- **阵列总长度**：约 128 × 0.8 mm = 102.4 mm

## 2. 闪烁体尺寸

### GOS闪烁体（高能闪烁体）
- **长度（X方向）**：0.72 mm（`GOSlength`）
- **宽度（Y方向）**：2.2 mm（`GOSwidth`）
- **厚度（Z方向）**：0.5 mm（`GOSthickness`）
- **材料**：Gd₂O₂S（GOS）
- **密度**：2.9 g/cm³
- **单个像素体积**：0.72 × 2.2 × 0.5 = 0.792 mm³
- **注释**：代码中注释说明"GOS薄膜的尺寸是102.5mm×2.2mm×0.5mm"（这可能是整个阵列的总长度）

### GGAG闪烁体（低能闪烁体）
- **长度（X方向）**：0.72 mm（`GGAGlength`）
- **宽度（Y方向）**：2.1 mm（`GGAGwidth`）
- **厚度（Z方向）**：3.3 mm（`GGAGthickness`）
- **材料**：Gd₃Ga₂Al₃O₁₂（GGAG）
- **密度**：6.63 g/cm³
- **单个像素体积**：0.72 × 2.1 × 3.3 = 4.9896 mm³
- **注释**：代码中注释说明"高能闪烁体GGAG尺寸是102.5mm×2.1mm×3.3mm"（这可能是整个阵列的总长度）

## 3. 铜板尺寸

- **长度（X方向）**：102.5 mm（`Copperlength`）
- **宽度（Y方向）**：102.5 mm（`Copperlength`，与长度相同）
- **厚度（Z方向）**：0.6 mm（`Copperthickness`）
- **材料**：铜（Cu）
- **体积**：102.5 × 102.5 × 0.6 = 6,300.375 mm³

## 4. 待测物体

- **类型**：球体（`G4Sphere`）
- **半径**：15 mm（`sphereRadius`）
- **材料**：磷酸钙（Ca₃(PO₄)₂）
- **密度**：2.9 g/cm³
- **体积**：(4/3) × π × 15³ ≈ 14,137.17 mm³
- **可移动**：可通过 `ObjShift` 参数在Y方向移动

## 5. 位置参数（距离）

### 射线源位置
- **射线源Z坐标**：-433 mm
  - 射线源位于 `G4ThreeVector(0*mm, 0*mm, -433.*mm)`
  - 粒子类型：γ射线（gamma）
  - 粒子能量：160 keV
  - 发射角度：扇形束
    - φ角度范围：-1° 到 +1°
    - θ角度范围：-7° 到 +7°

### 射线源到待测物体的距离
- **射线源到球中心距离**：433 mm
  - 从射线源（Z = -433 mm）到待测物体中心（Z = 0 mm）
  - 总距离 = 0 - (-433) = 433 mm

### 探测器到待测物体的距离
- **Distance_Object2GGAG**：92 mm
  - 从待测物体中心到GGAG闪烁体前表面的距离

### 闪烁体之间的间距
- **Distance_GGAG2Copper**：0.5 mm
  - GGAG闪烁体后表面到铜板前表面的距离

- **Distance_Copper2GOS**：3 mm
  - 铜板后表面到GOS闪烁体前表面的距离

### 位置计算（Z方向坐标）

以世界坐标系原点（0,0,0）为参考：

1. **射线源位置**：
   ```
   Z = -433 mm
   X = 0 mm
   Y = 0 mm
   ```

2. **待测物体中心位置**：
   ```
   Z = 0 mm（原点）
   Y = ObjShift（可调，默认0 mm）
   X = 0 mm
   ```

3. **GGAG闪烁体中心位置**：
   ```
   Z = Distance_Object2GGAG = 92 mm
   ```

4. **铜板中心位置**：
   ```
   Z = Distance_Object2GGAG - GGAGthickness/2.0 - Copperthickness/2.0 - Distance_GGAG2Copper
   Z = 92 - 3.3/2 - 0.6/2 - 0.5 = 92 - 1.65 - 0.3 - 0.5 = 89.55 mm
   ```

5. **GOS闪烁体中心位置**：
   ```
   Z = Distance_Object2GGAG - GGAGthickness/2.0 - Copperthickness - GOSthickness/2.0 
      - Distance_GGAG2Copper - Distance_Copper2GOS
   Z = 92 - 1.65 - 0.6 - 0.25 - 0.5 - 3 = 86.0 mm
   ```

## 6. 像素阵列排列

### X方向排列
- **像素索引**：0 到 127（共128个）
- **像素位置计算**：
  ```cpp
  G4int x_direction = NPx/2;  // = 64
  G4double Posx_shift = -(x_direction-i)*GapPeriod + GapPeriod/2.0;
  ```
- **中心像素位置**：X = 0 mm（当 i = 64 时）
- **最左侧像素**：X ≈ -51.2 mm（当 i = 0 时）
- **最右侧像素**：X ≈ +51.2 mm（当 i = 127 时）

### Y方向
- 所有像素的Y坐标均为 0 mm

## 总结

- **射线源**：γ射线，160 keV，位置Z = -433 mm
- **射线源到球距离**：433 mm
- **探测器类型**：双闪烁体阵列（GGAG + GOS）
- **像素数量**：128个
- **阵列总长度**：约102.4 mm
- **探测器到物体距离**：92 mm
- **待测物体**：半径15 mm的球体
- **几何结构**：从射线源到探测器依次为：射线源 → 待测物体 → GGAG → 铜板 → GOS
- **总距离**：射线源到GGAG探测器 = 433 + 92 = 525 mm
