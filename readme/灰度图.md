# X 射线双能分选图像处理流程文档

该流程主要用于将 Geant4 模拟生成的原始能量沉积数据（$81 \times 128$ 矩阵）转换为可用于智能分选的**透射率图像**与**材质识别 R 值图像**。

------

## 1. 第一阶段：标准化透射投影（Transmission Map）

**核心目标**：将“能量沉积绝对值”转换为物理意义明确的“透过率”，并消除硬件系统带来的非均匀性影响。

### 1.1 物理公式

$$T = \frac{I_{raw} - I_{dark}}{I_{flat} - I_{dark}}$$

- **$I_{raw}$**：扫描矿石得到的 $81 \times 128$ 原始能量沉积矩阵。
- **$I_{flat}$**：高统计量的“空扫”平场向量（$128$ 维），通过对多行空场数据取平均得到。
- **$I_{dark}$**：暗场底噪（在 Geant4 默认模拟中通常为 $0$）。

### 1.2 操作步骤

1. **分母准备**：取 81 行空扫数据的列平均值，生成平滑的“黄金平场线”。
2. **逐行归一化**：将原始矩阵的每一行点对点除以平场向量。
3. **灰度定标**：将范围在 $[0.0, 1.0]$ 的透过率 $T$ 乘以 $65535$，生成 **16-bit 标准灰度图**。

### 1.3 结果特征

- **消除几何效应**：解决了点源照射导致的“边缘暗、中间亮”问题。
- **可视化效果**：类似于医学 X 光片，图像灰度直接反映了物体的**面密度（厚度 × 密度）**。

------

## 2. 第二阶段：材质识别 R 值图像（Material/R-Map）

**核心目标**：消除厚度干扰，提取反映物质原子序数（$Z_{eff}$）的特征特征值，实现煤与矸石的分类。

### 2.1 物理公式

基于 Beer-Lambert 定律，物质对 X 射线的衰减（Attenuation）表示为：



$$A = -\ln(T) = \mu \cdot x$$

通过双能对比，定义 R 值：



$$R = \frac{A_{Low}}{A_{High}} = \frac{-\ln(T_{Low})}{-\ln(T_{High})} \approx \frac{\mu_{Low}}{\mu_{High}}$$

### 2.2 操作步骤

1. **对数变换**：对标准化后的低能（LE）和高能（HE）透射矩阵取负对数。

2. **安全处理**：为防止 `log(0)` 或负数，需进行 `np.where` 截断及 `np.abs` 处理。

3. 点对点相除：

   

   $$R = \frac{| \ln(T_L) | + \epsilon}{| \ln(T_H) | + \epsilon}$$

   

   其中 $\epsilon$ 为平滑因子（如你的代码中的 bix_1_value），用于稳定背景区域。

### 2.3 核心优势

- **厚度无关性（Thickness Independence）**：理论上，同一材质的 R 值与其厚度无关。在图像上表现为：同一种矿石无论薄厚，其 R 值灰度趋于一致。
- **材质敏感性**：低原子序数（煤）与高原子序数（矸石）在 R 值轴上具有显著的分离度。

------

## 3. 处理效果对比表

| **特性**     | **原始能量矩阵 (Raw)** | **标准透射图 (Transmission)** | **R 值材质图 (R-Map)**     |
| ------------ | ---------------------- | ----------------------------- | -------------------------- |
| **数值意义** | 能量沉积 (keV)         | 穿透率 (0% ~ 100%)            | 原子序数特征值 ($Z_{eff}$) |
| **几何影响** | 严重（边缘暗影）       | 已消除                        | 已消除                     |
| **厚度影响** | 随厚度增加而变暗       | 随厚度增加而变暗              | **理论无关（关键！）**     |
| **主要用途** | 数据采集、初步验证     | 形状识别、物体定位            | **智能分选、材质定性**     |

## 4. 算法实现代码建议 (Python)

Python

```
import numpy as np

def calculate_r_map(le_raw, he_raw, le_flat, he_flat):
    # 1. 平板校正 (线性空间)
    le_trans = le_raw / le_flat
    he_trans = he_raw / he_flat
    
    # 2. 转换到对数空间 (衰减空间)
    # 加上微小偏移防止 log(0)
    A_l = np.abs(np.log(le_trans + 1e-6))
    A_h = np.abs(np.log(he_trans + 1e-6))
    
    # 3. 计算 R 值 (材质空间)
    # bix_1_value 保证了背景空气区域 R 值稳定在 1.0 附近
    EPS = 1e-3
    r_map = (A_l + EPS) / (A_h + EPS)
    
    return r_map
```
