# CZT 探测器模拟系统说明

## 一、CZT 是什么？

### 1.1 CZT 材料定义

**CZT** 是 **碲锌镉**（Cadmium Zinc Telluride，化学式：Cd₁₋ₓZnₓTe）的缩写，是一种重要的半导体材料。

#### 基本特性：
- **化学组成**：由镉（Cd）、锌（Zn）和碲（Te）组成的化合物半导体
- **晶体结构**：闪锌矿结构（Zincblende）
- **密度**：约 5.8-6.0 g/cm³
- **原子序数**：高（Cd: 48, Te: 52），对高能光子有良好的吸收能力

#### 主要应用：
1. **X射线和γ射线探测器**
   - 直接转换型探测器（不需要闪烁体）
   - 高能量分辨率
   - 可在室温下工作（不需要液氮冷却）

2. **医学成像**
   - CT扫描
   - 核医学成像
   - 乳腺X射线成像

3. **安全检查**
   - 行李扫描
   - 货物检测

4. **科学研究**
   - 高能物理实验
   - 天体物理观测

#### CZT 探测器的优势：
- ✅ **高能量分辨率**：能量分辨率可达 1-2%（在 662 keV）
- ✅ **室温工作**：不需要冷却系统
- ✅ **高原子序数**：对高能光子吸收效率高
- ✅ **直接转换**：光子直接转换为电荷，减少信号损失
- ✅ **像素化阵列**：可以实现高空间分辨率

## 二、本项目中的 CZT 探测器模拟系统

### 2.1 项目概述

本项目是一个基于 **Geant4** 的 X 射线探测器模拟系统，虽然项目名称包含 "CZT"，但**实际使用的探测器材料是双闪烁体系统**：

1. **GGAG**（Gd₃Ga₂Al₃O₁₂）- 低能闪烁体
2. **GOS**（Gd₂O₂S）- 高能闪烁体

这种设计可能是为了：
- 实现能量分辨（低能和高能分别探测）
- 提高探测效率
- 优化不同能量范围的响应

### 2.3 工作原理

#### 模拟流程：

1. **粒子生成**（PrimaryGeneratorAction）
   - 生成 160 keV 的 γ 射线
   - 扇形束：φ = -1° 到 +1°，θ = -7° 到 +7°
   - 从 Z = -433 mm 位置发射

2. **物理过程**（PhysicsList）
   - 使用 Geant4 物理过程模拟
   - 包括：光电效应、康普顿散射、电子对产生等

3. **几何追踪**（SteppingAction）
   - 追踪粒子在探测器中的每一步
   - 记录能量沉积
   - 根据材料类型（GGAG 或 GOS）分别记录

4. **事件处理**（EventAction）
   - 每个事件结束时汇总能量沉积
   - 填充直方图
   - 累积到运行级别数据

5. **运行管理**（RunAction）
   - 管理整个运行过程
   - 创建输出文件
   - 保存结果到 ROOT 文件和文本文件

### 2.4 输出数据

#### 1. ROOT 文件（`Mydata.root`）
- **256 个直方图**：
  - Pixel NO. 1000-1127：GGAG 材料（128个像素）
  - Pixel NO. 2000-2127：GOS 材料（128个像素）
- **内容**：每个像素的能量沉积分布（0-200 keV，1000个bin）

#### 2. 文本文件
- **`LowEnergy/XX.X.txt`**：GGAG 闪烁体的总能量沉积（128个数值）
- **`HighEnergy/XX.X.txt`**：GOS 闪烁体的总能量沉积（128个数值）
- **文件名**：根据 `ObjShift` 参数命名（如 `0.0.txt`, `0.7.txt` 等）

### 2.5 主要功能

1. **步进扫描**
   - 通过 `scan_step.sh` 脚本实现
   - 自动改变待测物体位置（ObjShift）
   - 范围：-28.0 mm 到 +28.0 mm
   - 步长：0.7 mm
   - 每个位置运行指定数量的事件

2. **能量分辨**
   - GGAG：主要探测低能部分
   - GOS：主要探测高能部分（经过铜板过滤后）
   - 通过对比两个探测器的响应，可以分析能量分布

3. **空间分辨**
   - 128 个像素阵列
   - 像素间距：0.8 mm
   - 可以重建物体的空间分布

## 三、为什么使用双闪烁体系统？

### 3.1 设计理念

虽然项目名称是 "CZT"，但实际使用了 **GGAG + GOS 双闪烁体系统**，这种设计有以下优势：

#### 1. **能量分辨能力**
- **GGAG**（低能闪烁体）：
  - 厚度较大（3.3 mm）
  - 密度高（6.63 g/cm³）
  - 主要吸收低能光子
  
- **GOS**（高能闪烁体）：
  - 位于铜板之后
  - 铜板过滤掉低能光子
  - 主要探测高能光子

#### 2. **提高探测效率**
- 不同材料对不同能量范围的响应不同
- 双闪烁体可以覆盖更宽的能量范围
- 减少能量损失

#### 3. **材料特性**

**GGAG (Gd₃Ga₂Al₃O₁₂)**：
- 高密度（6.63 g/cm³）
- 高原子序数（Gd: 64）
- 良好的闪烁性能
- 快速衰减时间

**GOS (Gd₂O₂S)**：
- 中等密度（2.9 g/cm³）
- 高原子序数（Gd: 64）
- 高光输出
- 常用于医学成像

### 3.2 与 CZT 的对比

| 特性 | CZT 探测器 | 本项目（GGAG+GOS） |
|------|-----------|------------------|
| **类型** | 直接转换 | 间接转换（闪烁体） |
| **能量分辨率** | 1-2% | 取决于闪烁体 |
| **工作温度** | 室温 | 室温 |
| **像素化** | 是 | 是（128像素） |
| **成本** | 较高 | 相对较低 |
| **应用** | 高能物理、医学 | X射线成像、材料分析 |

## 五、技术栈

### 5.1 核心依赖

- **Geant4**：蒙特卡洛粒子物理模拟框架
- **ROOT**：数据分析框架（用于数据存储和分析）
- **CMake**：构建系统
- **C++**：编程语言

### 5.2 主要组件

1. **DetectorConstruction**：定义几何结构和材料
2. **PrimaryGeneratorAction**：定义粒子源
3. **PhysicsList**：定义物理过程
4. **SteppingAction**：追踪粒子步进
5. **EventAction**：处理每个事件
6. **RunAction**：管理运行和输出

## 六、总结

### CZT 材料
- **CZT** = 碲锌镉（Cd₁₋ₓZnₓTe）
- 重要的半导体探测器材料
- 用于 X 射线和 γ 射线探测
- 高能量分辨率，室温工作

### 本项目系统
- **项目名称**：CZT 探测器模拟系统
- **实际材料**：GGAG + GOS 双闪烁体系统
- **功能**：X 射线成像和材料分析
- **方法**：基于 Geant4 的蒙特卡洛模拟
- **特点**：双闪烁体能量分辨 + 像素化空间分辨

虽然项目名称包含 "CZT"，但实际实现使用了更灵活的双闪烁体设计，这种设计在某些应用场景下可能比单一 CZT 探测器更有优势，特别是在需要能量分辨和成本控制的场景。
